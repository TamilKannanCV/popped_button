# popped_button

This package creates a 3d popped button

## Preview
<img src="assets/gifs/preview.gif" width="200">


## Usage
```dart
PoppedButton(
    depth: 10.5,
    color: Colors.yellow,
    child: Center(
        child: Text(
            "Click me!",
            style: TextStyle(fontWeight: FontWeight.bold),
        ),
    ),
    onPressed: () {
        debugPrint("Pressed");
    },
)
```
